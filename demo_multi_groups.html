⚙️ Как это работает

1. Создание пользователя с несколькими группами:

{
    username: 'tillo',
    password: '123456',
    role: 'warehouse',
    warehouseGroup: ['Сугд', 'Хисор'],  // Массив групп!
    blocked: false
}

2. Фильтрация складов:

function filterWarehousesByUserGroups(warehouses) {
    if (currentUser && currentUser.role === 'warehouse' && currentUser.warehouseGroup) {
        const userGroups = Array.isArray(currentUser.warehouseGroup) ? 
            currentUser.warehouseGroup.filter(g => g !== '') : 
            [currentUser.warehouseGroup].filter(g => g !== '');
        
        if (userGroups.length > 0) {
            return warehouses.filter(w => userGroups.includes(w.group));
        }
    }
    return warehouses;
}

3. Получение названий складов пользователя:

function getUserWarehouseNames() {
    if (currentUser && currentUser.role === 'warehouse' && currentUser.warehouseGroup) {
        const userGroups = Array.isArray(currentUser.warehouseGroup) ? 
            currentUser.warehouseGroup.filter(g => g !== '') : 
            [currentUser.warehouseGroup].filter(g => g !== '');
        
        if (userGroups.length > 0) {
            return appData.warehouses
                .filter(w => userGroups.includes(w.group))
                .map(w => w.name);
        }
    }
    return appData.warehouses.map(w => w.name);
}

📋 Инструкция по настройке

Шаг 1: Создание групп складов

В разделе "Управление" → "Подгруппы складов" создайте нужные группы:

  • Сугд
  • Хисор
  • Душанбе
  • и т.д.

Шаг 2: Создание складов

В разделе "Управление" → "Склады" создайте склады и назначьте им группы:

  • Склад Сугд-1 → Группа: Сугд
  • Склад Хисор-А → Группа: Хисор

Шаг 3: Создание пользователя-завсклада

В разделе "Пользователи" создайте пользователя:

  • Логин: tillo
  • Роль: Завсклад
  • Подгруппы складов: Выберите несколько групп, удерживая Ctrl

Шаг 4: Проверка работы

Войдите под пользователем tillo и убедитесь, что:

  • В выпадающих списках складов отображаются только склады из выбранных групп
  • В отчетах показываются данные только по доступным складам
  • Нельзя создать операции для недоступных складов

✅ Преимущества новой системы

  • Гибкость: Завсклад может работать с несколькими группами одновременно
  • Безопасность: Доступ только к назначенным складам
  • Удобство: Простой интерфейс выбора групп
  • Масштабируемость: Легко добавлять новые группы и склады
  • Совместимость: Работает со старыми данными (одна группа)
